# Cloud-Based PostgreSQL and PostgreSQL-Compatible Databases

## PostgreSQL-Based (Direct or Compatible)

### Major Cloud Providers

- AWS RDS for PostgreSQL - Fully managed PostgreSQL database by Amazon
- AWS Aurora PostgreSQL - Amazon's high-performance, distributed PostgreSQL variant
- Azure Database for PostgreSQL - Microsoft's managed PostgreSQL service
    + Single Server (legacy)
    + Flexible Server (current)
- Google Cloud SQL for PostgreSQL - Google's managed PostgreSQL offering
- Google AlloyDB - Google Cloud's PostgreSQL-compatible analytical database

### Specialized/Niche Providers

- Neon - Serverless PostgreSQL with automatic scaling
- Supabase - Open-source Firebase alternative built on PostgreSQL
- Timescale Cloud - Managed TimescaleDB (PostgreSQL extension for time-series data)
- Tembo - PostgreSQL variants optimized for different workloads
- PgVector Cloud - PostgreSQL with vector database capabilities
- Crunchy Bridge - Managed PostgreSQL from the PostgreSQL experts
- Hydrozen - Distributed PostgreSQL platform
- Vercel Postgres - Serverless PostgreSQL powered by Neon

### Simpler/Lower-Cost Options

- Heroku Postgres - Salesforce-owned managed PostgreSQL
- DigitalOcean Managed Databases - PostgreSQL managed service
- Render - PostgreSQL hosting with simple deployment
- Railway - PostgreSQL with easy-to-use deployment
- Fly.io - PostgreSQL in edge infrastructure
- Linode Managed Database - PostgreSQL on Linode infrastructure
- Vultr Managed Database - PostgreSQL on Vultr infrastructure

## Key Features Most Share

- ACID transactions
- Complex queries and JOINs
- Extensibility (functions, triggers, custom types)
- JSON/JSONB support
- Full-text search
- Window functions
- Common Table Expressions (CTEs)
